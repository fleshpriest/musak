# Musak
Musak downloads music from YouTube playlists using yt-dlp.

## Quickstart
- run `pip install -r requirements.txt`
- Rename envCHANGEME.py to env.py
  - Replace all placeholder variables in env.py
- install [deno](https://deno.com/)

### Why Musak
- Handles multiple playlists
- Shorter subsequent run times for large playlists.
  - Filters out previous downloads before instructing yt-dlp to download
